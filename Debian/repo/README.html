Skyenet APT Repository

The Skyenet APT repository sources are found on GitHub at Skyb0rg007/packages.

Installation

To use this repository, you must first import the GPG key and APT sources file. This can be done manually, but I also provide a helper .deb package to install those files. It can be downloaded here. 

$ wget https://apt.soss.website/pool/main/s/skyenet-archive-keyring/skyenet-archive-keyring_1.0.0-1_all.deb
$ sudo apt install ./skyenet-archive-keyring_1.0.0-1_all.deb
$ sudo apt install <package>
